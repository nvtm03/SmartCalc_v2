Программа реализована на графической библиотеке QT. Калькулятор имеет способностью вычислять результат введенных выражений, включая тригонометрию и вычисление корня из числа. Также имеется возможность отображать график функции от X

Пользователь может составить введенное выражение. Будет построен график с значениями по умолчанию, где вы сможете просмотреть график функции несмотря на его размер, так как существует возможность масштабирования.

Install

Для того чтобы начать использовать калькулятор, его необходимо установить с помощью команды make install. Эта команда установит программу на рабочий стол в папку SmartCalc_v2.0.

Uninstall

Вы также можете набрать make uninstall, чтобы удалить приложение

ZIP

Также можно архивировать проект с помощью команды make dist.
